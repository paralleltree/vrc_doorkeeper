# vrc_doorkeeper

VRChatでのユーザ入退室をXSOverlayに通知するやつです。

![image](https://user-images.githubusercontent.com/7324519/145714476-15a2de78-460a-43ab-bbe4-a46d7ca35b51.png)

## 使い方
XSOverlay、VRChat(VRモード)を起動してから`vrc_doorkeeper.exe`を起動してください。

`vrc_doorkeeper.exe`はどこに置いても大丈夫です。

Windows Defenderなどウイルス対策ツールにてブロックされる場合は除外リストに追加してください。

黒い画面を閉じると終了します。
